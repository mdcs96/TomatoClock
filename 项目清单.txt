🚀 如何获取 APK:
---------------

方案1: GitHub Actions (推荐 ⭐⭐⭐⭐⭐)
  - 无需本地环境
  - 上传到 GitHub 自动编译
  - 约 10-15 分钟
  - 详见: 获取APK的三种方法.md

方案2: Android Studio (适合开发者)
  - brew install --cask android-studio
  - 打开项目编译
  - 约 20-25 分钟
  - 详见: 快速开始.md

方案3: 命令行编译 (需要 Android SDK)
  - ./setup_and_build.sh
  - 自动安装并编译
  - 约 12-15 分钟
  - 详见: README.md
