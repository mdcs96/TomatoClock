# 🍅 番茄钟 - 快速开始指南

## 最简单的编译方式

### 方式一：一键编译（推荐）

打开终端，运行：

```bash
cd ~/Desktop/TomatoClock
./setup_and_build.sh
```

脚本会自动：
1. 检查并安装 Gradle
2. 检查并安装 Android SDK
3. 编译生成 APK
4. 将 APK 复制到桌面

### 方式二：使用 Android Studio

1. **下载并安装 Android Studio**
   - 官网: https://developer.android.com/studio
   - 或使用命令: `brew install --cask android-studio`

2. **打开项目**
   - 启动 Android Studio
   - 选择 "Open" → 选择 `~/Desktop/TomatoClock` 文件夹
   - 等待 Gradle 同步完成（首次可能需要几分钟）

3. **编译 APK**
   - 菜单栏: Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 等待编译完成
   - 点击弹出的通知中的 "locate" 查看 APK

### 方式三：手动命令行编译

如果已经安装了 Android SDK：

```bash
cd ~/Desktop/TomatoClock

# 设置 Android SDK 路径
export ANDROID_HOME=$HOME/Library/Android/sdk

# 使用 Gradle 编译
gradle wrapper
./gradlew assembleDebug

# APK 生成位置
ls -lh app/build/outputs/apk/debug/app-debug.apk
```

## 安装到手机

### Android 手机

1. **传输 APK**
   - 通过 USB 数据线连接手机
   - 将 `TomatoClock.apk` 复制到手机
   - 或通过微信、QQ 等发送给自己

2. **安装**
   - 在手机上打开文件管理器
   - 找到 `TomatoClock.apk`
   - 点击安装
   - 如提示"不允许安装未知应用"：
     * 进入"设置" → "安全"
     * 允许"未知来源"或"安装未知应用"
     * 选择文件管理器，允许安装

3. **使用**
   - 打开"番茄钟"应用
   - 选择模式（工作/短休息/长休息）
   - 点击"开始"按钮
   - 享受专注时光！

## 应用功能

- **🍅 工作模式**: 25分钟专注工作
- **☕ 短休息**: 5分钟快速恢复
- **🌙 长休息**: 15分钟深度放松
- **⏯️ 控制**: 开始、暂停、重置
- **📳 提醒**: 计时结束震动提醒

## 番茄工作法建议

1. **选择任务** - 确定要完成的工作
2. **开始番茄钟** - 设置25分钟专注工作
3. **专注工作** - 避免一切干扰
4. **短暂休息** - 完成后休息5分钟
5. **重复循环** - 每4个番茄钟后长休息15分钟

## 常见问题

**Q: 为什么编译失败？**
A: 确保已安装 Java JDK 和 Android SDK，运行 `./setup_and_build.sh` 自动安装。

**Q: 可以修改时长吗？**
A: 可以！编辑 `MainActivity.java` 文件中的时间常量：
```java
private static final long WORK_TIME = 25 * 60 * 1000; // 改这里
private static final long SHORT_BREAK_TIME = 5 * 60 * 1000;
private static final long LONG_BREAK_TIME = 15 * 60 * 1000;
```

**Q: 如何添加声音提醒？**
A: 需要在 `res/raw/` 目录添加音频文件，并在 `onFinish()` 方法中播放。

**Q: 支持后台运行吗？**
A: 当前版本需要保持应用在前台。可以通过添加 Service 实现后台运行。
