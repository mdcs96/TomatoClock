## 🚀 如何获得 APK

### 方案 A: 使用 GitHub Actions (推荐，最简单)

1. **上传到 GitHub**
   ```bash
   cd ~/Desktop/TomatoClock
   git init
   git add .
   git commit -m "番茄钟应用"
   git branch -M main
   # 创建 GitHub 仓库后运行
   git remote add origin https://github.com/你的用户名/TomatoClock.git
   git push -u origin main
   ```

2. **等待自动编译**
   - GitHub Actions 会自动开始编译
   - 约 5-10 分钟后完成

3. **下载 APK**
   - 进入仓库的 "Actions" 标签
   - 点击最新的工作流
   - 下载 Artifacts 中的 APK

### 方案 B: 本地编译（需要 Android SDK）

#### B1. 使用 Android Studio
```bash
# 1. 安装 Android Studio
brew install --cask android-studio

# 2. 打开 Android Studio
# 3. Open Project → 选择 ~/Desktop/TomatoClock
# 4. Build → Build Bundle(s) / APK(s) → Build APK(s)
```

#### B2. 使用命令行（需要先安装 Android SDK）
```bash
cd ~/Desktop/TomatoClock
./setup_and_build.sh
```

或手动执行：
```bash
# 安装 Android SDK
brew install --cask android-commandlinetools

# 设置环境变量
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin

# 安装必要组件
sdkmanager --install "platform-tools" "platforms;android-34" "build-tools;34.0.0"

# 编译
cd ~/Desktop/TomatoClock
./gradlew assembleDebug

# APK 位置
ls -lh app/build/outputs/apk/debug/app-debug.apk
```

## 📱 如何安装到手机

1. **传输 APK 到手机**
   - USB 数据线连接
   - 或通过微信/QQ发送给自己
   - 或使用 AirDrop (iPhone → Android 不支持)

2. **在手机上安装**
   - 打开文件管理器
   - 找到 `TomatoClock.apk` 或 `app-debug.apk`
   - 点击安装
   - 如提示"不允许未知来源"，去设置中允许

3. **开始使用**
   - 打开"番茄钟"应用
   - 选择模式（工作/短休息/长休息）
   - 点击"开始"
