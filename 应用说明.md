# 🍅 番茄钟应用说明

## 应用界面

```
┌─────────────────────────┐
│     🍅 番茄钟            │
│                         │
│                         │
│       25:00             │ ← 大号计时器显示
│                         │
│                         │
│  ┌─────┐ ┌─────┐ ┌─────┐ │
│  │开始 │ │暂停 │ │重置 │ │ ← 控制按钮
│  └─────┘ └─────┘ └─────┘ │
│                         │
│      选择模式            │
│                         │
│  ┌─────┬─────┬─────┐    │
│  │工作 │短休息│长休息│    │ ← 模式切换
│  │25分 │5分  │15分 │    │
│  └─────┴─────┴─────┘    │
└─────────────────────────┘
```

## 颜色方案

- **主色调**: 番茄红 (#D84315)
- **背景色**: 浅米黄 (#FFF5E6)
- **绿色按钮**: 开始 (#4CAF50)
- **橙色按钮**: 暂停 (#FF9800)
- **红色按钮**: 重置 (#F44336)
- **蓝色按钮**: 短休息 (#2196F3)
- **紫色按钮**: 长休息 (#9C27B0)

## 功能特性

### 1. 计时功能
- 精确的倒计时
- 大号数字显示（分:秒格式）
- 实时更新，每秒刷新

### 2. 三种模式
- **工作模式**: 25分钟，适合专注工作
- **短休息**: 5分钟，适合快速恢复
- **长休息**: 15分钟，适合深度放松

### 3. 控制功能
- **开始**: 启动当前选择的计时模式
- **暂停**: 暂停计时，保留剩余时间
- **重置**: 将计时器重置为模式初始值

### 4. 提醒功能
- 计时结束时震动1秒
- 需要震动权限（已在 Manifest 中声明）

## 技术实现

### 核心技术
- **CountDownTimer**: Android 系统倒计时器
- **Vibrator**: 震动服务
- **ConstraintLayout**: 响应式布局
- **Material Design**: 现代化UI设计

### 权限说明
```xml
<uses-permission android:name="android.permission.VIBRATE" />
<uses-permission android:name="android.permission.POST_NOTIFICATIONS" />
```

- `VIBRATE`: 用于计时结束时的震动提醒
- `POST_NOTIFICATIONS`: 未来可扩展通知功能

### 兼容性
- 最低支持: Android 7.0 (API 24)
- 目标版本: Android 14 (API 34)
- 屏幕方向: 仅竖屏模式

## 使用场景

- 📝 写作和创作

- 🎨 设计绘画
- 📊 数据分析

### 番茄工作法流程
1. **准备阶段**: 列出今天要完成的任务
2. **第一个番茄钟**: 选择一个任务，开始25分钟
3. **短休息**: 完成后休息5分钟，喝水、伸展
4. **继续循环**: 重复2-3步
5. **长休息**: 每完成4个番茄钟，休息15-30分钟

### 提高效率的建议
- 📵 关闭手机通知
- 🚫 避免社交媒体
- 🎧 可以听白噪音或专注音乐
- 📝 记录完成的番茄钟数量
- 🎯 每个番茄钟只做一件事
